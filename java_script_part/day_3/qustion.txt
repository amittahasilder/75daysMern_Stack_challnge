🔥 1. JavaScript Operators (All Topics)
✅ 1. Arithmetic Operators

+ - * / % **

Example:
let a = 10, b = 3;

console.log(a + b); // 13
console.log(a % b); // 1
console.log(a ** b); // 1000
Interview Q:

👉 What is % ?
✔️ Answer: It returns the remainder.

✅ 2. Assignment Operators

= += -= *= /=

let x = 5;
x += 3; // 8
Q:

👉 Difference between = and ==?
✔️ = assign, == compare

✅ 3. Comparison Operators

== === != !== > < >= <=

console.log(5 == "5");  // true
console.log(5 === "5"); // false
Q:

👉 Difference between == and ===?
✔️ === checks value + type

✅ 4. Logical Operators

&& || !

let age = 20;
console.log(age > 18 && age < 30); // true
Q:

👉 What is ||?
✔️ OR operator

✅ 5. Unary Operators

++ -- typeof

let x = 5;
x++; // 6
console.log(typeof x); // number
✅ 6. Ternary Operator
let age = 18;
let result = (age >= 18) ? "Adult" : "Minor";
✅ 7. Type Operators

typeof, instanceof

console.log(typeof "Hello"); // string
🔥 2. Conditionals (All Topics)
✅ 1. if Statement
if (age > 18) {
  console.log("Adult");
}
✅ 2. if...else
if (age > 18) {
  console.log("Adult");
} else {
  console.log("Minor");
}
✅ 3. else if Ladder
let marks = 80;

if (marks >= 90) {
  console.log("A+");
} else if (marks >= 70) {
  console.log("A");
} else {
  console.log("Fail");
}
✅ 4. switch Case
let day = 2;

switch(day) {
  case 1:
    console.log("Monday");
    break;
  case 2:
    console.log("Tuesday");
    break;
  default:
    console.log("Invalid");
}
🔥 3. Important Interview Questions + Answers
💡 Q1:

👉 What is truthy & falsy?
✔️ falsy: 0, "", null, undefined, NaN, false

💡 Q2:

👉 What will this return?

console.log(0 || "Hello");

✔️ Answer: "Hello"

💡 Q3:

👉 What is short-circuiting?
✔️ Logical operators stop early.

💡 Q4:

👉 Output?

console.log(5 > 3 ? "Yes" : "No");

✔️ Yes

💡 Q5:

👉 Difference between switch & if?
✔️ switch = fixed values
✔️ if = complex logic
